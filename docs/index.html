Features

⚡

Fast XOR Encryption

NumPy-powered pixel operations process images instantly using bitwise XOR with a key-derived keystream.

🔄

Symmetric

Same key encrypts and decrypts. XOR is its own inverse — no separate decrypt algorithm needed.

🖼️

Multi-Format

Supports PNG, JPEG, BMP, and TIFF input. Always outputs lossless PNG for perfect decryption.

📊

Key Strength Meter

Real-time feedback on your key quality — length, character variety, and pattern detection.

👁️

Realtime Preview

Watch the encryption process unfold chunk by chunk with the realtime preview mode.

📁

Batch Processing

Encrypt or decrypt entire folders of images at once with the same key.

🌐

Browser Version

No installation needed. The web version runs entirely client-side — your images never leave your device.

✓

Verification

Built-in round-trip verification confirms decryption matches the original image exactly.

How It Works

1

Key → Seed

Your text key is hashed with SHA-256 and converted to a deterministic 32-bit seed.

seed = SHA256(key) mod 2³²
2

Seed → Keystream

A pseudorandom number generator produces a keystream matching the image dimensions (H × W × 3 bytes).

keystream = PRNG(seed) → [0-255] for each pixel channel
3

XOR

Each pixel's RGB values are XOR'd with the corresponding keystream values, scrambling the image.

encrypted_pixel = original_pixel ⊕ keystream_pixel
4

Decrypt

Apply the same key again. XOR is self-inverse, so the original image is restored.

original = encrypted ⊕ keystream = (original ⊕ keystream) ⊕ keystream
